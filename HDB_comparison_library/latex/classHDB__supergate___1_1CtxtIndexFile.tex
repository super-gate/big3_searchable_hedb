\doxysubsection{Detailed Description}
class representing a ciphertext index file 

A ciphertext intex file is a collection of Ctxt\+Indexes. It is a collection of $<$column name, \mbox{\hyperlink{classHDB__supergate___1_1CtxtIndex}{Ctxt\+Index}}$>$ pairs, with the column name and corresponding ciphertext index as a pair.
